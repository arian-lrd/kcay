# Frontend

This folder will contain the frontend application.

When you're ready to set up the frontend, you can:

- **React**: Run `npx create-react-app .` (when in this directory)
- **Vite + React**: Run `npm create vite@latest . -- --template react`
- **Next.js**: Run `npx create-next-app@latest .`
- **Vue**: Run `npm create vue@latest .`
- Or any other frontend framework of your choice
