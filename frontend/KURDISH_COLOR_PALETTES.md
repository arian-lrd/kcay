# Kurdish-Inspired Color Palettes for KCAY

Based on the Kurdish flag and cultural symbolism, here are color palette options for your student club website.

## üé® Color Symbolism

The Kurdish flag features these meaningful colors:
- **Red** (#ED2024): Blood of martyrs, struggle for freedom and dignity
- **Green** (#278E43): Beauty of Kurdistan, life and vitality
- **Yellow/Gold** (#FEBD11): Source of life and light, the sun (Newroz symbol)
- **White** (#FFFFFF): Peace and equality
- **Turquoise**: Cultural significance (stones from sky, blessings)

---

## üåü Palette Option 1: Traditional Red & Green (Recommended)

**Best for:** Strong, proud Kurdish identity

```css

--nav-text: #FFFFFF;          /* White */
--nav-hover: #ED2024;         /* Bright Kurdish Red */
--nav-active: #FEBD11;        /* Kurdish Yellow/Gold accent */
--nav-border: #9B0E1F;        /* Darker red border */

```

**Why this works:** Red is the most prominent color in the Kurdish flag, representing strength and identity.

## üéØ How to Apply a Palette

1. Open `frontend/src/app/globals.css`
2. Find the section: `/* Navigation Header Colors - Easy to customize! */`
3. Replace the color values with your chosen palette
4. Save the file - changes apply instantly!

### Example (Option 1 - Traditional Red & Green):

```css
:root {
  --background: #ffffff;
  --foreground: #171717;
  
  /* Navigation Header Colors - Easy to customize! */
  --nav-bg: #C8102E;           /* Deep Kurdish Red */
  --nav-text: #FFFFFF;          /* White */
  --nav-hover: #ED2024;         /* Bright Kurdish Red */
  --nav-active: #FEBD11;        /* Kurdish Yellow/Gold accent */
  --nav-border: #9B0E1F;        /* Darker red border */
  --nav-mobile-bg: #9B0E1F;     /* Dark red mobile menu */
}
```
